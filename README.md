#  Student Information System

A desktop-based Student Information System developed using **C# (.NET Windows Forms)** to manage student records, courses, enrollments, and administrative tasks in an organized and efficient manner.

---

##  Project Overview

The Student Information System (SIS) is designed to help universities and educational institutions manage academic data digitally.  
It replaces manual record-keeping with a centralized software solution that improves efficiency, accuracy, and accessibility.

---

##  Objectives

The main objectives of this project are to:

- Automate student and course management processes  
- Provide an easy-to-use interface for administrators  
- Store and manage student academic information securely  
- Reduce paperwork and human errors  
- Generate useful reports for decision making  

---

##  Problem Statement

Many institutions still rely on manual or outdated systems to manage:

- Student records  
- Course information  
- Enrollments  
- Grades  
- Administrative tasks  

This leads to:

- Data inconsistency  
- Loss of records  
- Slow processing  
- Inefficiency  

This application solves these issues by providing a structured digital platform.

---

##  Technologies Used

- **Programming Language:** C#
- **Framework:** .NET 6 Windows Forms
- **Database:** SQLite / SQL Server (Configurable)
- **IDE:** Visual Studio
- **Architecture:** Layered Architecture
- **Version Control:** Git & GitHub

---

##  Features

###  Authentication
- User login system  
- Secure access to dashboard  

###  Student Management
- Add new students  
- Update student details  
- Delete student records  
- View all students  

###  Course Management
- Add new courses  
- Edit course information  
- Remove courses  
- View course list  

###  Enrollment System
- Enroll students in courses  
- View enrollments  
- Manage academic records  

###  Dashboard
- Overview of:
  - Total students  
  - Total courses  
  - Total enrollments  

###  User-Friendly Interface
- Clean and modern UI  
- Easy navigation  
- Organized menu system
